## 1. if 문 기본 구조
```java
if (조건문) {
    <수행할 문장1>;
    <수행할 문장2>;
    ...
} else {
    <수행할 문장A>;
    <수행할 문장B>;
    ...
}
```

## 2. 비교연산자
|비교연산자|설명|
|:---:|:---:|
|x < y|x가 y보다 작다|
|x > y|x가 y보다 크다|
|x == y|x와 y가 같다|
|x != y|x와 y가 같지 않다|
|x >= y|x가 y보다 크거나 같다|
|x <= y|x가 y보다 작거나 같다|
* 예시
  <br>

  ```java
  int x = 3;
  int y = 2;
  System.out.println(x != y);
  ```
  ```java
  true
  ```

## 3. and(&&), or(||), not(!)
|연산자|동작|
|:---:|:---:|
|<code>x && y</code>|x와 y 모두 참이어야 참이다|
|<code>x \|\| y</code>|x와 y 둘 중 적어도 하나가 참이면 참이다|
|<code>!x</code>|x가 거짓이면 참이다|
